# FinalJayjay - Otomasi Testing Web & API
 
Framework otomasi testing untuk website DemoBlaze dan API di https://dummyapi.io/data/v1/.  
Menggunakan Java, Selenium WebDriver, RestAssured, Cucumber (BDD), dan Gradle.

## Tools  
- Java  
- Selenium WebDriver  
- RestAssured  
- Cucumber  
- Gradle  
- WebDriverManager
